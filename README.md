# AgriGuard - Intelligent Precision Farming System

**Lead Developer:** Soumoditya Das  
**Contact:** soumoditt@gmail.com  
**Event:** Kshitij 2026 Hackathon - Precision Agriculture Track

AgriGuard is a next-generation PWA (Progressive Web App) that leverages multimodal AI to democratize agronomy for smallholder farmers. It combines satellite-based field intelligence, real-time computer vision for pest diagnostics, and a vernacular voice coach into a single, cohesive platform.

## ðŸš€ Key Innovations

*   **Multilingual AI Agronomist**: A voice-first interface supporting regional Indian languages (Hindi, Tamil, Telugu, Bengali, Marathi) to bridge the literacy gap.
*   **Field Architect & Health Maps**: 
    *   Satellite-powered boundary mapping.
    *   **NDVI Health Simulation**: Visualizes crop stress zones using simulated multispectral analysis.
    *   **IoT Grid Deployment**: Simulates sensor network placement for moisture and NPK monitoring.
*   **Crop Doctor (Computer Vision)**: 
    *   Instant detection of pests (e.g., Fall Armyworm) and diseases.
    *   **Lifecycle Analysis**: Identifies larval stages to recommend precise IPM strategies.
*   **Soil Health Lab**: AI-driven analysis of soil parameters to generate NPK dosage reports.
